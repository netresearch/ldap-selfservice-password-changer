# LDAP Selfservice Password Changer

## Developing

### Updating the styles or HTML

Prerequisites:

- Node.js v16+
- PNPM

```bash
# Install dependencies
pnpm i

# Build once
pnpm css

# Build and watch for changes
pnpm css:dev
```
